Formatos de imagens

-São padrões de codificação, compressão, e definição que determina o tipo de arquivo gráfico, como exemplo temos:

• .BMP[Windows Bitmap]
O formato é basicamente um arquivo que descreve quantos pixels a imagem tem e qual é a cor de cada pixel. Os valores de cada luz variam de 0 a 255 porque cada um deles têm 8 bites (ou um byte).
Ou seja, cada pixel de uma imagem BMP tem três bytes.

• .TIFF[Tagged Image Format]
-É usado extensamente em aplicações de manipulação de imagens tais como Photoshop, DTP e scanners.

• .GIF[Graphics Interchange Format]
Surgiu antes do PNG, com a intenção de suprir a deficiência da compressão JPG para arquivos gráficos. É o melhor formato para imagens em movimento

• .SVG[Scalable Vector Graphics]
-É um arquivo vetorial criado especialmente para a web. Funciona tanto com imagens estáticas, quanto com animações em 2D.

• .ICO[Icon]
Um arquivo ICO é um formato de imagem usado para armazenar ícones de programas, arquivos e pastas do Windows. Ele contém dois bitmaps: 1) o bitmap AND - a máscara de imagem (que determina qual parte do ícone é transparente) e
2) o bitmap XOR - contém o ícone que é mapeado na máscara de imagem.

• .JPG[Joint Photographics Experts Group]
Diferente do formato BMP, ele divide a imagem em blocos de 8 por oito pixels e compara cada um a um dos 64 padrões existentes. Em seguida, determina qual o “peso” de cada um desses padrões em cada bloco.

• .PNG[Portable Networks Graphics]
-Criado para facilitar a troca de imagens pela internet, assim como o JPG, esse formato usa compressão para reduzir o peso de arquivos.
O formato reúne características do JPG e do GIF, porém com opções mais avançadas. Não preserva o fundo.

• .GIF[Graphics Interchange Format].EPS
-Pode ser usado para armazenar gráficos. Formato utilizado principalmente quando há recortes ou traçados, além de seu uso para imagens vetoriais e bitmap.

As TAGS que usamos para as Imagens

Um exemplo de como anexei a imagem anterior:

• As tags "figure" e "figcaption" foram definidas a partir do HTML 5.

No campo "src" é onde você colocará a origem, ou a fonte da imagem que voce quer dar uploud. O "width" é a largura da imagem e o "height" a altura.

Sintaxe: < figure >.....< /figure >
Legenda da imagem dentro do figure:
< figcaption >legenda< /figcaption >
