HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criadas na década de 1990 por Tim Berners-Lee.

A HTML foi criada para publicação de conteúdo em hipertexto, tais como:

  • texto;
  • imagem;
  • áudio
  • vídeo
Estes elementos podem ser vinculados através de hyperlinks.

World Wide Web

TAGS

• As tags definem ao navegadora marcação de conteúdo.

• Tem funcionalidades específicas. Sem o use de tags o navegador exibe um texto plano.

• Podem possuir atributos, ou seja, são parametros que definem algum tipo de informação extra ao navegador.
